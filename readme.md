### Installation

1. Clone this repository:
   ```
   git clone https://github.com/Shyurer/account-discord-online-forever
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Fill in your information in the `config.json` file:
   ```
   {
     "tokens": [
       "your_token"
     ]
   }
   ```
4. Run the application:
   ```
   node index.js
   ```
